Missing cases

Information missing on NOMIS or NDelius

Make sure information about the case has been entered correctly on NOMIS. For example, this might include sentence calculation dates.

Also check if the case is listed on Add missing details. If it is, you will need to manually add some information from NDelius before it can be displayed.

Check the case needs a POM

Cases that do not need a POM will not be displayed. This might be because someone:

  • is on remand
  • is under 18
  • is involved in a civil case
  • does not have an earliest release date or parole hearing date in NOMIS

Life sentences

If someone on a life sentence is showing as a determinate case, you need to make the case active on NOMIS.
